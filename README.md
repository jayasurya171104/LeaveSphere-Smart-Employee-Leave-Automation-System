
# LeaveSphere-Smart-Employee-Leave-Automation-System

🚀 LeaveSphere – Smart Employee Leave Automation System

📌 Project Overview

LeaveSphere is a web-based **Employee Leave Management System** designed to automate and simplify the leave request and approval process within an organization.

The system allows employees to apply for leave, track their leave status, and enables team leaders and administrators to efficiently manage and approve leave requests.

This system improves organizational workflow, reduces manual work, and provides clear visibility of employee leave information.

** ✨ Key Features**

 **👨‍💼 Employee Module**
- 🔐 Secure Login
- 📝 Apply Leave
- 📋 View My Leaves
- 📅 Leave Calendar
- 📊 Leave Balance Tracking
- 🧑‍💻 Personal Dashboard
- 👤 Profile Management
- 🌙 Dark / ☀️ Light Theme Support

 **🧑‍💼 Team Leader Module**
- 📊 Team Leader Dashboard
- 👥 View Department Employees
- 📋 View Department Leave Requests
- ✅ Approve / ❌ Reject Employee Leaves
- 📝 Apply Personal Leave
- 📅 Leave Calendar
- 📈 Leave Status Summary

 **🛠 Admin Module**
- 📊 Admin Dashboard
- 👥 Manage Employees
- 🏢 Manage Departments
- 👨‍💼 Manage Team Leaders
- ✅ Approve Team Leader Leaves
- 📑 View Employee Leave Reports
- 📑 View Team Leader Leave Reports
- 📊 Department-wise Leave Reports

**🔄 System Workflow**

1️⃣ Employee logs into the system  
2️⃣ Employee applies for leave  
3️⃣ Leave request goes to the **Team Leader of that department**  
4️⃣ Team Leader **approves or rejects** the leave  
5️⃣ If the Team Leader applies leave → **Admin reviews it**  
6️⃣ Admin **approves or rejects Team Leader leave**  
7️⃣ All leave data is tracked in dashboards and reports
